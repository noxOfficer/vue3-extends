# Vue 3 - extends example

This example will show you a simple Vue 3 Option API component reusable structure to extends a component('TAG') with new functionalities keeping the original ones
Sounds strange but this have some different implementation issues like "merge"

> A Vue.js project

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
